listとしてアクセス

  • listの最初(0番目){{ input_from_python[0] }}
  • 次(1番目){{ input_from_python[1] }}
  • 非常に固定的

    • listとしてアクセス（forを使う方法）

    {% for value in input_from_python %}
  • {{ value }}
    • {% endfor %}

    listとしてアクセス（forを使う方法）した後に、 さらにlistとしてアクセス

    {% for value in input_from_python %} {% for i in range(2) %}
  • {{ value[i] }}
    • {% endfor %} ---
    {% endfor %}
